Introducción

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear páginas web. Es fundamental para entender cómo funciona la estructura de una página web.

Estructura básica

Una página HTML básica consta de elementos esenciales como <html>, <head>, <title> y <body>. Estos elementos proporcionan la estructura y el contenido de una página web.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
</head>
<body>
    Contenido de la página...
</body>
</html>

Elementos y etiquetas

Los elementos HTML se componen de etiquetas que rodean el contenido. Por ejemplo, <h1> es una etiqueta para los encabezados de nivel 1, <p> para los párrafos, y así sucesivamente.

Ejemplo práctico

A continuación, mostraremos un ejemplo práctico de una página HTML básica:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi primera página web</title>
</head>
<body>
    <header>
        <h1>Bienvenidos a mi página web</h1>
    </header>
    <main>
        <p>Este es un párrafo de ejemplo.</p>
    </main>
</body>
</html>
